# Creating an Office Addin using Angular2, TypeScript and Express

1. Create the directory structure 
    ```
    └── src
    ├── client
    │   ├── app
    │   ├── images
    │   └── js
    ├── server
    └── shared
    ```
2. Run npm init to initialize a project
3. Create a .gitignore file
4. Run git init to initialize a repository 
5. Inital Checkin
